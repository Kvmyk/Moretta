MORETTA

Use Claude, ChatGPT, or Gemini at work without exposing confidential data. Moretta anonymises your files locally, sends only a masked version to AI, then restores your real data into the result. Your data never leaves.

How it works

Four steps.
Zero leaks.

Confidential data never crosses the network boundary. The AI sees only anonymous tokens — never the names, PESELs, or contracts beneath them.

01
Upload
Employee drops a DOCX, XLSX or email into the local web UI
→
02
Mask
Local model detects PII, replaces with UUID tokens. Vault stores the mapping.
→
03
AI Task
Masked file sent to Claude / GPT-4o / Gemini with the employee's instruction
→
04
Restore
Real data re-injected locally from vault. Employee receives the final file.

Installation

One command.
Fully running.

Runs entirely on your own infrastructure. No cloud. No SaaS. No data leaving your network. Requires Docker and 8 GB RAM minimum.

# clone the repository
git clone https://github.com/Kvmyk/moretta.git
cd moretta
cp .env.example .env

# add your AI provider API keys to .env, then:
docker-compose up
